# 🎨 Design System — Tatá Teruya

## Color Palette

| Token             | Value      | Usage                  |
|--------------------|------------|------------------------|
| `--primary`        | `#C4956A`  | Buttons, accents       |
| `--primary-dark`   | `#A67B52`  | Prices, hover states   |
| `--pix-green`      | `#32BCAD`  | Pix buttons            |
| `--bg`             | `#FAF7F4`  | Page background        |
| `--bg-footer`      | `#3D2E24`  | Footer                 |
| `--text-primary`   | `#2C1E14`  | Headings               |
| `--text-secondary` | `#6B5B4F`  | Body text              |
| `--success`        | `#5A9A6E`  | Confirmation messages  |

## Typography

- **Display**: Playfair Display (headings, logo)
- **Body**: Inter (body text, buttons, labels)

## Spacing

8px system: `4 / 8 / 16 / 24 / 32 / 48 / 64px`

## Components

- **Product Card**: rounded corners, border, shadow on hover, image zoom on hover
- **Pix Modal**: backdrop blur, bounce-in animation, QR code + copy key
- **WhatsApp Float**: pulsing green button, fixed bottom-right
- **Toast**: slide-up notification for copy confirmation

## Responsiveness

| Breakpoint | Columns | Target     |
|------------|---------|------------|
| < 600px    | 1       | Mobile     |
| ≥ 600px    | 2       | Tablet     |
| ≥ 960px    | 3       | Desktop    |
| ≥ 1200px   | 4       | Wide       |
